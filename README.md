[![sites](docs/RP2040.png)](https://www.raspberrypi.org/documentation/rp2040/getting-started/)

#### 关键特性

* 264KB RAM, 128K-256K Flash
* up to 16MB QSPI Flash
* 2 × UART, 2 × SPI , 2 × I2C
* USB1.1 OTG with UF2
